# Giza World

AI 3D Exhibition — A browser-based 3D Gaussian Splatting (3DGS) scene viewer and character animation platform.

## Features

- **3D Scene Viewer** — Load PLY/SPZ Gaussian splatting scenes with two-scene switching
- **Character Animation** — Display rigged 3D characters with multi-animation switching
- **Interactive Controls** — Drag to orbit, scroll to zoom, drag characters to reposition
- **Text-to-Motion (T2M) Panel** — API-ready interface for future motion generation integration

## Tech Stack

- [Three.js](https://threejs.org/) — WebGL 3D rendering engine
- [Spark](https://sparkjs.dev/) (World Labs) — Gaussian splatting renderer
- Tailwind CSS — UI styling
